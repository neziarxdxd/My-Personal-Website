EVENT REGISTRATION APPLICATION

Created by Programmer ako -_-

This application helps the event organizer to make their registration for event easier and more secured. It also helps to make the attendance process system as easy as possible, it also provide onsite registration, event maker, QR generator for attendance, and QR Scanner for attendance that suitable in any camera, and also a certificate generator to complete the attendance

MODULES

  • xlsxwriter
  • pyzbar
  • opencv-python
  • sqlite3
  • reportlab
  • qrcode
  • pillow
  • openpyxl
  • tkcalendar

✅ EXCEL to DATABASE

I used xlrd python module for this feature of the application
PROBLEM: most of the registration are online based and the application that I made is a desktop application made in python
SOLUTION: Create a feature that can import the excel file and transfer the data to the database with the help of xlrd module and SQLite3 module

✅ DATABASE to EXCEL

I used XlsxWriter for this features
this can help the event organizer to print the data in the database

✅ EVENT CRUD (Create,Read, Update, Delete)

it has create event feature wherein you can create your own event it has the event name, event date, event venue You can also update it and delete it

✅ REGISTRATION CRUD (Create,Read, Update, Delete)

This feature is for the onsite registration , for example if the attendees wants to edit their names, organization and more
